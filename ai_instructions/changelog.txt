Victoria 3 Timeline v1.0.2

Bug Fixes
- Fixed culture merge homeland remapping so merged cultures only inherited states that were already homelands of the source culture.

Dynamic Countries
- Added dynamic country identity events that let player-created subjects choose colonial, homeland, or multicultural culture policies and then customize their official religion, while AI-created countries weight both choices by citizenship and church laws.
- Added a Dynamic Country Builder toggle that lets players automatically apply law-based culture and religion identity choices instead of answering the identity events.
- Adjusted Millet System auto-religion behavior so dynamic countries prefer their local homeland faith instead of automatically inheriting the creator's state religion.
- Removed the automated law-following choice from the player religion popup so manual dynamic-country religion choices are explicit.
- Added visible state-religion effect previews to each manual dynamic-country religion choice.
- Weighted the player religion popup recommendations so the checkmarked option matches the creator's church law, including State Religion recommending preservation of the creator's faith.
- Simplified manual dynamic-country religion choices to preserve the creator's state faith, adopt a local popular faith, or proclaim state atheism.
- Added explicit Subjecthood citizenship handling so dynamic countries favor homeland-based subject governments over colonial-only or multicultural identities.

AI
- Added Lite and On modes for AI Improvement, with Lite preserving the yearly economy pass and On running it monthly.
